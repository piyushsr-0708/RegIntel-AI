TROUBLESHOOTING
================================================================================

Backend won't start:
→ Activate venv: venv\Scripts\activate
→ Install deps: pip install -r requirements.txt

Login fails with 401:
→ Check credentials match defaults
→ Verify database seeded: sqlite3 data/compliance.db "SELECT * FROM users;"

Department sees no assignments:
→ HEAD_OFFICE must assign requirements first
→ Check assignments table: sqlite3 data/compliance.db "SELECT * FROM assignments;"

JWT token expired:
→ Token expires after 8 hours
→ User must login again

Dependencies missing:
→ Reinstall: pip install fastapi uvicorn sqlalchemy python-jose passlib python-multipart
