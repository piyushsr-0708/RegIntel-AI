OBLIGATION TYPE CLASSIFICATION (Priority-based)
------------------------------------------------
Priority 1: Conditional (if sentence starts with "if", "where", etc.)
Priority 2: Prohibited ("shall not", "must not")
Priority 3: Mandatory ("shall", "must", "required to")
Priority 4: Recommended ("should", "advised to")
Priority 5: Conditional (fallback)
Default: Informational

EFFECTIVE STATUS DETECTION
---------------------------
- Superseded: Contains "superseded", "replaced", "no longer applicable"
- Proposed: Contains "proposed", "draft", "consultation"
- Active: Default for all others

import json
with open('requirements/requirements_taxonomy.json', 'r', encoding='utf-8') as f:
    data = json.load(f)

# Filter by domain
kyc_reqs = [r for r in data if r['domain'] == 'KYC']

# Get mandatory requirements
mandatory = [r for r in data if r['obligation_type'] == 'Mandatory']

# Find requirements with deadlines
with_deadlines = [r for r in data if r['deadline']]

# Group by source document
from collections import defaultdict
by_source = defaultdict(list)
for req in data:
    by_source[req['source_document']].append(req)

# Find prohibited actions
prohibited = [r for r in data if r['obligation_type'] == 'Prohibited']
